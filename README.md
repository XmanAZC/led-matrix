# LED Matrix项目实现

## 概述

项目有如下几个功能：

1. NTP+RTC，实现时间的校准。
2. WiFi配网。
3. MIC采集+FFT转换。
4. W2812b矩阵屏幕显示。
5. BLE通信控制，w2812b显示亮度滚动文字等。
6. OTA升级，功能。
